# Simple Tools (シンプルツールズ)

「速さ」「安全性」「使いやすさ」を追求した、ブラウザ完結型のWebツールプラットフォームです。

## 🚀 特徴

- **インストール不要**: すべてのツールがブラウザ上で動作します。
- **プライバシー保護**: ほとんどのツールがクライアントサイド（ローカル）で処理されるため、データがサーバーに送信されることはありません。
- **洗練されたデザイン**: 「Soft Space」をテーマにした幻想的な宇宙背景と、ふわふわと浮かぶアンチグラビティ・デザイン。
- **軽量・高速**: 余計な依存関係を排除し、高速な読み込みと操作感を実現。

## 🛠 搭載ツール一覧

### テキスト・開発系
- **文字数カウンター**: リアルタイムな文字数・行数カウント。
- **URLエンコード/デコード**: 日本語URLの作成やパラメータ解析。
- **Base64変換**: テキスト・バイナリの相互変換 (UTF-8対応)。
- **JSONフォーマッター**: 整形、バリデーション、簡易修復。
- **HTMLエンティティ変換**: 特殊文字のエスケープ・アンエスケープ。
- **SQLフォーマッター**: SQLクエリの整形・ハイライト。
- **JSON to Go Struct**: JSONデータからGo言語の構造体を自動生成。
- **テキスト比較 (Diff)**: 2つのテキストの差異を行単位で表示。
- **Markdownプレビュー**: リアルタイム変換・HTMLコピー。
- **正規表現チェッカー**: JS正規表現のテスト・ハイライト。
- **CSS Minifier**: CSSの圧縮・軽量化。
- **CSV to RAG**: LLMのRAG用に構造化されたMarkdownへの変換。

### セキュリティ・認証系
- **JWT Debugger**: JWTトークンのデコード・内容確認。
- **UUID生成器**: v4 UUIDの一括生成・コピー。
- **ハッシュ生成・比較**: SHA-256/512, HMACの生成。
- **パスワード生成器**: 強力なランダムパスワードの一括生成 (全角対応)。

### 便利ツール・解析系
- **UNIX時間変換**: タイムスタンプ(秒/ミリ秒)と日時の相互変換。
- **単位変換ツール**: 長さ、重さ、データ、Web(px/rem)の変換。
- **Metaタグ生成**: SEO/OGP/Twitter Cardの一括生成。
- **PDFテキスト抽出**: PDFからテキストデータを安全に抜き出し。
- **MIDIデータ解析**: MIDIバイナリ構造の可視化。
- **日本祝日カレンダー**: 祝日・振替休日の自動計算表示。
- **同時ページビューアー**: 複数URLの同時表示・比較。
- **クライアント情報表示**: ブラウザやシステム環境の表示。

### 画像・制作系
- **QRコード生成**: カラー・サイズ・誤り訂正レベル設定可能なQR作成。
- **QRコードリーダー**: カメラを使用したリアルタイムQRスキャン。
- **絵文字ジェネレーター**: Slack/Discord用のカスタム絵文字作成。
- **カラーピッカー**: Web制作向けの色抽出・見本作成。
- **画像リサイズ**: ブラウザ内でのリサイズ・変換。

## 📂 各フォルダの構成

```text
.
├── assets/                 # 共通アセット
│   ├── css/                # 全ページ共通のグローバルスタイル
│   └── js/                 # 各ツールのロジックJS
├── [tool-name]/            # 各ツールの個別ディレクトリ (index.html)
├── index.html              # ポータル（ホームページ）
├── about.html              # サイト概要（AdSense最適化済み）
├── privacy.html            # プライバシーポリシー
└── README.md               # このドキュメント
```

## 🛠 使用技術

- **Frontend**: HTML5, Vanilla CSS3 (Soft Space Theme), JavaScript (ES6+)
- **Libraries**:
  - `QRious`: QRコード生成
  - `pdf.js`: PDF解析
  - `@tonejs/midi`: MIDI解析
- **API**: Web Crypto API (セキュアなハッシュ/パスワード生成)
- **Analytics**: Google Analytics 4
